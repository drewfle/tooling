# Configs

## Sh

In `~/.zshrc`:

```sh
# macOS
source $HOME/.dev/tooling/configs/sh/macos.zshrc
# Linux
source $HOME/.dev/tooling/configs/sh/linux.liushpc-nix.zshrc
# Linux WSL
source $HOME/.dev/tooling/configs/sh/linux-wsl.zshrc
```

## Git

In `~/.gitconfig`:

```gitconfig
[include]
  path = ~/.dev/tooling/configs/git/.gitconfig
```
